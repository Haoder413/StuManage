# 学生课程排课联动设计

## 目标

学生新增时可以选择一门课程，并为这名学生设置固定上课星期、开始时间和结束时间。保存学生后，系统同时创建学生-课程关联和固定排课，排课考勤日历自动显示这条课程安排。

## 范围

- 只实现新增学生时的课程选择和固定时间段。
- 不做课程默认时间。
- 不做学生详情页编辑课程。
- 不改课程创建流程。

## 数据流

1. 新增学生页加载课程列表。
2. 用户填写学生信息，选择课程、星期、开始时间、结束时间。
3. 前端提交 `courseId`、`dayOfWeek`、`startTime`、`endTime`。
4. 学生接口创建学生。
5. 如果选择了课程，接口创建 `StudentCourse`。
6. 如果同时有课程和时间，接口创建 `Schedule`，类型为 `fixed`。
7. 排课日历继续通过现有 `/api/schedules` 读取数据并显示。

## 验证

- 新增学生接口支持保存课程和固定排课。
- 新增学生页包含课程和固定时间字段。
- 构建通过。
