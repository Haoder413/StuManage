# 家长个人安排批量重复与小卡片设计

日期：2026-06-24

## 背景

家长端时间管理已经支持查看老师课程和创建家长自己的个人安排。现在需要增强个人安排：

- 新增安排时可以一次选择多个星期，例如每周一、每周二。
- 可以设置截止日期，系统批量生成符合条件的安排。
- 个人安排在日历中用预设图标或小卡片展示，而不是只显示普通文字或圆点。
- 批量生成的安排后续编辑和删除按整组处理，不按单条处理。

## 目标

- 家长新增个人安排时支持重复星期多选。
- 家长选择重复星期后必须填写截止日期。
- 系统按开始日期、重复星期和截止日期批量生成个人安排。
- 家长可以为个人安排选择预设图标小卡片。
- 日历中个人安排显示为带图标和颜色的小卡片。
- 批量生成的个人安排共享同一个分组，编辑或删除其中任意一条时影响整组。
- 老师安排保持只读蓝色课程样式，不参与个人图标选择。

## 非目标

- 不支持上传自定义图片。
- 不做“只改这一次 / 以后所有 / 全部”分支。
- 不做冲突检测。
- 不影响老师端排课、考勤、课时统计、课堂反馈和学习进度。

## 数据设计

扩展 `ParentScheduleItem`：

- `seriesId`：同一批重复安排共用的分组标识。普通单条安排为空。
- `iconKey`：预设图标类型。
- `seriesEndDate`：重复安排截止日期。普通单条安排为空。
- `repeatDays`：重复星期快照，保存为 JSON 字符串。普通单条安排为空。

预设图标先使用固定 key：

- `math`：数学
- `english`：英语
- `homework`：作业
- `reading`：阅读
- `music`：音乐
- `sports`：运动
- `other`：其他

每个 key 在前端映射到固定图标、短标签和颜色。

## 创建规则

新增个人安排表单包含：

- 学习关系
- 标题
- 开始日期
- 开始时间
- 结束时间
- 备注
- 预设图标
- 重复星期多选
- 截止日期

如果没有选择重复星期：

- 创建一条普通个人安排。
- `seriesId`、`seriesEndDate`、`repeatDays` 为空。
- `iconKey` 使用用户选择值，默认 `other`。

如果选择了重复星期：

- 截止日期必填。
- 截止日期不能早于开始日期。
- 从开始日期到截止日期逐日扫描。
- 日期星期命中所选重复星期时创建一条个人安排。
- 所有生成记录使用同一个 `seriesId`。
- 所有生成记录保存相同 `iconKey`、`seriesEndDate` 和 `repeatDays`。
- 如果开始日期本身星期不在所选重复星期中，不创建开始日期当天记录。

## 编辑规则

编辑普通个人安排：

- 只更新当前记录。

编辑重复安排中的任意一条：

- 以这条记录的 `seriesId` 为准，更新同一 `seriesId` 下当前家长自己的全部记录。
- 标题、学习关系、时间段、备注和图标同步更新到整组。
- 如果修改开始日期、重复星期或截止日期，采用删除旧组再重新生成新组的方式，避免残留旧日期。

## 删除规则

删除普通个人安排：

- 只删除当前记录。

删除重复安排中的任意一条：

- 删除同一 `seriesId` 下当前家长自己的全部记录。
- 确认文案说明会删除整组安排。

## 接口设计

`/api/parent/schedule-items` 扩展：

### GET

个人安排返回新增字段：

- `seriesId`
- `iconKey`
- `seriesEndDate`
- `repeatDays`

### POST

新增请求字段：

- `iconKey`
- `repeatDays`
- `seriesEndDate`

服务端根据是否传入 `repeatDays` 决定创建单条或批量创建。

### PATCH

如果目标记录有 `seriesId`：

- 删除旧 series 下当前家长自己的全部记录。
- 按新表单参数重新生成整组。

如果目标记录没有 `seriesId`：

- 更新当前单条记录。

### DELETE

如果目标记录有 `seriesId`：

- 删除同一 series 下当前家长自己的全部记录。

如果没有 `seriesId`：

- 删除当前单条记录。

所有写操作继续校验：

- 当前用户必须是家长。
- 学生必须属于当前家长。
- 学习关系必须属于当前家长且和学生一致。
- 只能操作当前家长自己的个人安排。

## 页面设计

新增个人安排弹窗增加：

- 图标小卡片选择区。
- 重复星期多选区。
- 截止日期输入框。

显示规则：

- 未选择重复星期时，截止日期可以为空。
- 选择任意重复星期后，截止日期变为必填。
- 重复安排详情显示重复规则，例如“每周一、周二，至 2026-08-31”。

日历展示：

- 月视图个人安排显示预设小卡片：图标 + 简短标题。
- 周视图个人安排显示图标 + 标题 + 时间。
- 老师安排继续使用蓝色老师课程样式。

## 验证

- 检查 schema 包含 `seriesId`、`iconKey`、`seriesEndDate`、`repeatDays`。
- 检查创建单条个人安排仍可用。
- 检查选择周一和周二，并设置截止日期后，批量生成对应日期记录。
- 检查重复安排任意一条编辑后，整组被更新。
- 检查重复安排任意一条删除后，整组被删除。
- 检查个人安排在月视图和周视图显示预设小卡片。
- 检查老师安排样式和只读行为不受影响。
- 检查构建通过。
