# 课堂回顾与标签设计

## 背景

排课考勤页当前点击出勤、请假、补课会直接写入考勤记录。新需求是在出勤或补课时补充本次课堂信息，并在学生详情页回顾历史上课内容。

## 设计

- `Attendance` 增加 `lessonContent`、`lessonFeedback`、`contentTags`、`feedbackTags` 字段。
- 新增 `LessonTag` 表，按 `type` 区分 `content` 和 `feedback` 两类标签。
- 排课考勤页点击 `出勤` 或 `补课` 时打开课堂反馈弹窗，填写上课内容、上课反馈，并用 chips 选择/新增内容标签和反馈标签。
- `请假` 保持快速标记，不要求填写课堂反馈。
- 学生详情页新增“历史上课内容回顾”，按日期倒序展示出勤/补课记录、上课内容、上课反馈和标签 chips。
- 标签管理放在课堂反馈弹窗内：可新增、重命名、删除标签；删除标签不会改写已有历史记录里的文本快照。

## 接口

- `/api/attendance` 接收课堂内容、反馈和标签数组，创建考勤记录后返回完整记录。
- `/api/lesson-tags` 提供标签列表、新增、编辑、删除。

## 验证

- 用脚本检查 Prisma schema 包含课堂字段和标签表。
- 用脚本检查考勤接口保存课堂内容与标签。
- 用构建检查页面类型与服务端渲染。
